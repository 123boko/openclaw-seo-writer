# Tools

## web_search
- Use for keyword research, SERP analysis, and competitor content review
- Search for "People Also Ask" questions related to the target keyword
- Look up current search intent and top-ranking content structure

## browser
- Use to visit and analyze top-ranking pages for the target keyword
- Extract content structure, word count, headings, and content gaps
- Check featured snippets and SERP features for the keyword

## read
- Use to read existing content files provided by the user
- Read reference materials, brand guidelines, or style guides
- Read previously written articles to maintain voice consistency

## write
- Use to save completed articles as markdown files
- Save outlines, drafts, and final versions separately when requested
- Always save to the location specified by the user

## exec
- Use sparingly and only when needed for content-related tasks
- Can be used to run local SEO tools or scripts if available
